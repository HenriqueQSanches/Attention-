Arraste para posicionar. Toque num móvel para selecionar: aí dá pra trazer pra frente (▲), mandar pra trás (▼) ou remover (✕).

Adicionar móvel
Só um protótipo pra sentir a interação: sem colisão, sem grade, posição e ordem livres. No app de verdade, cada móvel vira um { id, x, y, z } salvo no banco, e o z é a ordem que eu escolho.
